CodeForces 919D Substring

发表于 2018-02-22 | |

http://codeforces.com/contest/919/problem/D

Tag: DP/拓扑排序

题意

一个有向图，每个结点存一个字符，路的长度定义为路径上出现最多的字符数，如”abaca”的长度为3，求最大路径长度。

注意：允许自环、多重边，图不一定连通。

思路

DP+拓扑排序。

dp[i][j] 表示以顶点i为终点的所有路径，字符j出现的最大次数。

使用拓扑排序遍历图，deg[i]记录i的入度，初始将所有入度为0的点加入队列，当队列不空时进行以下操作：

  • 取队首顶点v，变量cnt++
  • 遍历以v为起点的边，设终点为k，用dp[k][0...25]更新 dp[v][0...25]
  • deg[v]减去1，若等于0，则将v加入队列

当该图无环时，最终cnt=n，此时输出最大的f[i][j].

否则该图存在环，答案会是无穷大，此时输出-1
