Let’s play a game

时间限制: 1000 ms 内存限制: 65536 kb

总通过人数: 207 总提交人数: 244

题目描述

Nova君和LaoWang决定一分胜负。给定两个正整数a,b。Nova君和LaoWang轮流从中将较大的数字减去较小数字的整数倍（1倍，2倍等等）。并且保证每次减完不会出现负数的情况。由Nova君先手。最终在自己回合将其中一个数变为0的一放获胜。两个人智商都还行，都会采取最优策略，谁会赢呢？

输入

多组测试数据。对于每组测试数据，给出两个数字a和b（保证Int范围内）

输出

对于每组数据，输出获胜者的名字。

解题思路

一、问题分析

由题意可看出，当给出a和b后，胜负就已经确定了（至于为什么，还没有考虑，挖个坑），此时的局势分为两种：

  • 必胜态：在此状态下行动的玩家必胜
  • 必负态：在此状态下行动的玩家必负

并且还有如下性质：

  • 若一个状态下一步可转移到必负态，则此状态为必胜态。（即如果我能让对手进入必负态，那么我一定会这样做，这也就是题目中“两个人的智商都还行，都会采取最优策略”的内涵）。
  • 因为只有必胜、必负两种状态，故若一个状态下一步无法转移到必负态，则此状态为必负态。
  • 若一个状态下一步必定转移到必胜态，则此状态为必负态。（因为下一步对手肯定会赢）。

所以如果能够枚举当前状态的所有变化，就能判断出当前状态是必胜态还是必负态。

二、递归解法

使用递归来求解当前状态的所有变化。

设当前状态为a和b (保证a $\ge$ b)，构造递归判断函数judge(int a, int b) 来执行判断。

递归方法如下：

  1. 若b $=$ 0

    则此状态为必负态，因为上一个人已经把两个数中的一个减成0了。

  2. 若b $\ne$ 0

    则该状态可能是必胜态，也可能是必负态，需要对下一步所有可能的状态进行枚举。

    对于

    这所有的可能性，只要其中有必负态，则当前状态为必胜态；若所有都是必胜态，则当前状态为必负态。故用judge函数判断这每一个状态，就能判断当前状态了。
