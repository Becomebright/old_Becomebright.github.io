日常陷入迷茫

发表于 2017-11-19 | |

这周很难受。

周初是腰疼，尤其周二跑完1000米，晚上把腰给扭了，疼的都弯不了腰，躺了一晚上才好些。

腰不疼了，紧接着又是脖子疼，大概是因为每天躺在床上看手机的缘故吧。

身体不舒服，心情也不畅快，不想学习，不知道自己能干嘛。

面对未来，有期待，却没有去追求的决心。

想成为探索世界的秘密的人，却因自己智商的不足和身体的病痛而绝望着。

靠，想那么多干嘛魂淡。

早睡早起，每天跑步，别玩游戏，别看手机不就好了么！

继续加油~

“I can do all things!”
