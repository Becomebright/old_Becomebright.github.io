2016级算法第三次上机解题报告

Nov.13, 2017

一、总结

本次上机主要考查了动态规划问题，包括ALS、背包问题等。

动态规划问题求解的核心在于找到最优子结构，之后构造转移方程，上机过程中，由于A、B、F题较为基础，没花多大功夫就求得了转移方程，但对于C、D题却没有好的思路。后来经过助教提醒，才发现这两道题目都可以转换成背包问题求解，对于背包问题我们都很熟悉，但是却无法灵活的举一反三，这是本次上机我发现的问题，以后做题一定要活用已经掌握的知识。

二、解题报告

A Bamboo的小吃街

思路分析

这道题就是经典的ALS问题。

假设离开左右两边第1~i-1家店铺的最早时间均已求出，分别记为dp[1][1…i-1]和dp[2][1…i-1]，由于左边第i加店铺只能从左边第i-1家或右边第i-1家到达，那么到达这间店铺的最早时间就是dp[1][i-1]与dp[2][i-1]+t[2][i-1]之间的最小值，因为要在这件店铺停留p[1][i]时间，那么离开这间店铺的最早时间可表示为: dp[1][i] = min(dp[1][i-1], dp[2][i-1] + t[2][i-1]) + p[1][i]。同理离开右边第i家店铺的最早时间为： dp[2][i] = min(dp[1][i-1] + t[1][i-1], dp[2][i-1]) + p[2][i]。

则吃完这条小吃街的最短时间就为min(dp[1][n], dp[2][n])

从而可以看出，此问题具有最优子结构，且求dp[1][i]和dp[2][i]时都用到了dp[1][i-1]和dp[2][i-1]，即存在重叠子问题，因此是动态规划问题，可以使用memorized recursion或bottom-top方法求解。

算法分析

第i个状态可由第i-1个状态求出，时间复杂度为O(n)

C AlvinZH的奇幻猜想——三次方

思路分析

本题题意为，给一个正整数n，将它分成x个正整数的立方和，求x的最小值。

可将此问题转化为完全背包问题，所谓完全背包问题，是指有N种物品和一个容量为V的背包，每种物品都有无限件可用。第i件物品的费用是c[i]，价值是w[i]。­求解将哪些物品装入背包可使价值总和最大。

将此问题看作一个容量为V=n的背包，有N种物品，费用分别为1，价值均为-1，求当把背包正好装满时的最大价值总和Max。这样-Max即为装入背包的物品个数，也就是x的最小值。

完全背包问题的求解，可将第i种物品拆成V/c[i]件物品，即将问题转化成0-1背包，求解思路见下一题。这种方法的时间复杂度大于O(NV)

其中f[v]为前i种物品装入容量为v的背包能获得的最大价值，子问题分为不放该件物品，即f[v]；和放入该件物品，即f[v-cost]+w[i]，这样当v递增时，第i种物品可放入多件，满足完全背包的要求。

D 双十一的抉择

思路分析

本题题意为有n袋糖，每袋糖装的糖果数量Ni不同，在不拆开包装的条件下尽量将糖均分，输出两份糖的数量差。

此题可转化成0-1背包问题，所谓0-1背包问题，是指有N件物品和一个容量为V的背包。第i件物品的费用是c[i]，价值是w[i]。求解将哪些物品装入背包可使价值总和最大。对于本题，相当于将n个价值和花费均为Ni的物品装入容量为sum/2的背包，这样在最大化价值总和的同时，相当于尽量填满了这个背包。

对于0-1背包问题，子问题是前i个物品放入容量为v的背包能得到的最大价值f[i][v]。

可求得转移方程：f[i][v] = max{f[i-1][v], f[i-1][v-c[i]] + w[i]}, 含义为只考虑第i件物品放或不放，若不放则问题转化为求前i-1件物品放入容量为v的背包能获得的最大价值，即f[i-1][v]; 若放则问题转化为求前i-1件物品放入容量为v-c[i]的背包能获得的最大价值再加上w[i]，即f[i-1][v-c[i]] + w[i]。

算法分析

0-1背包问题要求得f[N][V]数组中的每个值，求一个的复杂度为O(1)，故时间复杂度为O(NV)，空间复杂度为O(NV)，经优化后可达到O(V)

E ModricWang's Polygons

思路分析

此题考察几何，题意为，在二维平面上有n个点，坐标均为整数，求这n个点能构成多少不同的正多边形。

首先需要明确整数点能构成的正多边形只能是正方形，这点直观上比较容易猜到，就不进行证明了。那么题目转化为求n个点能构成多少个不同的正方形。对与这n个点，可以每次枚举两个点，这两个点能构成的正方形只有两种情况，如图，分别求出两种情况下，另外两个点的坐标，再查找是否存在即可。对于n个点能构成的正方形，每条边上的两个点都会被枚举一次，于是得到的结果是所求正方形数的四倍。

算法分析

每次枚举两个点，进行两次for循环，复杂度为O(n^2)

F ModricWang的导弹防御系统

思路分析

这道题是最长递减子序列问题

设序列用数组a[]存储，dp[i]表示以a[i]为最小值的递减子序列的长度的最大值，那么转移方程为： dp[i] = max{dp[j]+1 | if a[j]>=a[i], for all j<i}

它的含义是， 以a[i]为最小值的递减序列长度，等于以a[j]为倒数第二小值的递减序列的长度+1，而这个长度是子问题dp[j]的值。

算法分析

对于每个dp[i]，要遍历dp[0 to i-1]才能求得，故时间复杂度为O(n^2)

G Winter is coming

思路分析

设数组dp[i][j][k]表示队伍中有i个士兵，j个野人，且队尾为士兵(k=0)或野人(k=1)时的排列种数。选择这样的子问题，可以保证士兵数<=n，野人数<=m，而连续的人数可在转移方程中体现。

dp[i][j][0]可分解成子问题：i个士兵、j个野人且队尾有L(1<=L<=min{x, i})个连续的士兵，求得转移方程：dp[i][j][0] = Σdp[i-L][j][1] for L=1 to min{x, i}

同理得dp[i][j][1] = Σdp[i][j-L][0] for L=1 to min{y, j}

初始时dp[0][0][0/1] = 1

最终答案为dp[n][m][0] + dp[n][m][1]

算法分析

三层循环，时间复杂度O(n*m*max{x, y}) = O(4*10), 空间复杂度O(2*n^2) = O(8*10)
